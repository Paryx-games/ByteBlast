# ByteBlast
A custom built Electron soundboard built for Windows

This is currently developed and maintained by **one person!** Therefore, updates can be slow.

What we currently support?

| Feature         	| Supported? 	|
|-----------------	|------------	|
| File Upload     	| ✅          	|
| Mic reroute     	| ✅          	|
| Spotify Support 	| ❌          	|
